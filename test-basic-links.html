Basic Link Test

This page tests if basic HTML links work without any JavaScript or complex CSS.

Instructions: Click each link below. They should either navigate to a new page or open in a new tab.

Test Links:

Google (External - New Tab) Contact Page (Internal) Services Page (Internal) Example.com (External - New Tab)

What Should Happen:

  • Google link: Opens Google in a new tab
  • Contact link: Navigates to contact.html
  • Services link: Navigates to services.html
  • Example link: Opens example.com in a new tab

If Links Don't Work:

Possible Issues:

  1. Browser Problem: Try a different browser or incognito mode
  2. JavaScript Blocked: Check if JavaScript is disabled
  3. Network Issue: Check your internet connection
  4. Security Software: Check if antivirus/firewall is blocking
  5. Browser Extensions: Disable ad blockers or other extensions

Quick Tests:

  • Right-click links: Try "Open in new tab" or "Open in new window"
  • Copy link address: Right-click and copy, then paste in address bar
  • Check console: Press F12 and look for errors
