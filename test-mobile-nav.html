Mobile Navigation Test

This page tests the mobile navigation using includes.js. Resize your browser to mobile size to see the hamburger menu.

If you can see the hamburger menu (three horizontal lines) on mobile, click it to test the mobile menu functionality.

Test Instructions:

  1. Resize your browser window to mobile size (width ≤ 768px)
  2. You should see a hamburger menu (three horizontal lines) in the top right
  3. Click the hamburger menu to open the mobile navigation
  4. The menu should slide in from the left
  5. Click on dropdown items to test dropdown functionality

Expected Behavior:

  • Hamburger menu visible on mobile (≤768px)
  • Desktop navigation hidden on mobile
  • Mobile menu slides in from left when hamburger clicked
  • Dropdowns work on mobile
  • Menu closes when clicking links
